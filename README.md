# React Native Swipeable POC

## How to run?

1. Download or clone this repo.

2. Install dependencies.

```js
yarn install
```

3. Navigate to `/ios` and run `pod install`.

4. Use `yarn ios` or `yarn android` to run on the perspective platforms.
